SKILLS: CAD, Arduino, Iterative Design, 3D Printing

Electronics

This electromechanical system was a monster to set up. I really wanted each joint to be individually controllable, which ment wiring up 15 solenoid valves both the the microcontroller and the air compressor - one for each of the 3 joints on each of the fingers.

Future Goals



Something I experimented with while working on this project was using 3D sculpting to create a much more organic finger shape. Though we tested some demos, the added material to make it look more realistic made the range of motion much smaller than the CADed version so the idea ended up being scrapped. However, I would love to explore the idea of combining 3D sculpting and traditional 3D modelling together in the future!
